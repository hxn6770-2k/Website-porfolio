K-Means Clustering Implementation

` The artifact aligns with the Category One: Software Engineering and Design. This project originates from a case study on user identification using smartphone accelerometer data, specifically focusing on walking activity. The dataset, titled "User Identification from Walking Activity Data," was collected from 22 individuals using an Android smartphone placed in their chest pockets during various outdoor activities. The primary algorithm employed is K-means clustering, a robust and widely-used unsupervised learning technique, which enables the segmentation of accelerometer data into distinct clusters representing different motion patterns.

How the Artifact Was Improved

To address the limitations of the original implementation, I made several key enhancements to improve the effectiveness and efficiency of the clustering process.

  1. K-means++ Initialization:
    • I replaced the random initialization of centroids with the K-means++ initialization method. This strategic selection of initial cluster centers reduces the likelihood of poor clustering results and improves algorithm convergence. As a result, the updated artifact now produces more accurate and consistent clusters.
    • This modification highlights my ability to make algorithmic design decisions that balance the trade-off between computational complexity and clustering performance.

    Here is the updated code for K-means++ initialization: 

                    
                // Updated code for K-means++ initialization

  2. Elbow Method for Optimal Cluster Selection:
    • I introduced the elbow method to systematically determine the optimal number of clusters (k). The elbow method calculates the within-cluster sum of squares (WCSS) for a range of k values, identifying the point where adding more clusters no longer significantly improves model performance.
    • This enhancement prevents both overfitting and underfitting, leading to more accurate clustering and effective modeling. It showcases my ability to implement data-driven approaches to enhance the algorithm’s effectiveness.

    The following code implements the elbow method for optimal cluster selection: 

                    
                # Function to find the optimal number of clusters using the elbow method
               find_optimal_k <- function(interests, max_k = 20) {
                  # Calculate WCSS for different values of k
                  wss <- numeric(max_k)
                  
                  for (k in 1:max_k) {
                    kmeans_result <- kmeans(interests, centers = k, nstart = 25)
                    wss[k] <- kmeans_result$tot.withinss
                  }
                  
                  # Find the elbow point (sharp decrease in WCSS)
                  elbow_point <- which.max(diff(diff(wss))) + 1
                  
                  return(elbow_point)
                }
  3. Silhouette Analysis for Cluster Evaluation:
    • I integrated silhouette analysis to evaluate the quality of the clusters. Silhouette values provide insights into how well each data point fits within its assigned cluster, compared to neighboring clusters. This helps assess cohesion and separation, improving the interpretability of the clustering outcomes.
    • This addition demonstrates my ability to use innovative tools to assess and optimize the clustering algorithm's performance, ensuring that it aligns with both computational principles and user needs.

    Below is the code for silhouette analysis: 

                    
                # Function to evaluate clustering quality using silhouette score
                evaluate_clustering <- function(kmeans_result, data, participant_index) {
                  cluster_assignments <- kmeans_result$cluster
                  dist_matrix <- dist(data)
                  dist_matrix_square <- as.matrix(dist_matrix)
                  silhouette_values <- numeric(length(cluster_assignments))
                  
                  for (i in 1:length(cluster_assignments)) {
                    cluster <- cluster_assignments[i]
                    same_cluster_distances <- dist_matrix_square[i, cluster_assignments == cluster & seq_along(cluster_assignments) != i]
                    a <- if (length(same_cluster_distances) > 0) mean(same_cluster_distances) else 0
                    b <- min(sapply(unique(cluster_assignments), function(c) {
                      if (c != cluster) mean(dist_matrix_square[i, cluster_assignments == c]) else Inf
                    }))
                    silhouette_values[i] <- if (max(a, b) > 0) (b - a) / max(a, b) else 0
                  }
                  
                  silhouette <- mean(silhouette_values, na.rm = TRUE)
                  wcss <- kmeans_result$tot.withinss
                  
                  return(list(participant_index = participant_index, silhouette = silhouette, wcss = wcss))
                }
